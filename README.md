# 8-bit AON Computer Simulator

Симулятор компьютера, который я разрабатываю с нуля. AON означает, что процессор проектируется из логических вентилей AND, OR, NOT.

Вы можете увидеть как он работет [здесь](https://).

Симулятор написан на языке Java Script (с оформлением интерфейса на HTML и CSS), чтобы каждый мог легко запустить его в своем web-браузере.

## Характеристики компьютера
- CPU 8-bit DATA, 16-bit ADDRESS;
- 4 общих 8-битных регистра;
- RAM, ROM, VRAM 64KB;
- Graphics 64-color 160x120 px, 26x15 text;
- Keyboard.
